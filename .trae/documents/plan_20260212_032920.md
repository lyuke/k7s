## K8s 资源管理增强计划

### 1. 扩展后端 API（kube.ts）
- 引入更多 K8s API：`AppsV1Api`（Deployment, DaemonSet, StatefulSet, ReplicaSet）、`BatchV1Api`（Job）、`BatchV1beta1Api`（CronJob）
- 新增类型定义：
  - `DeploymentInfo`、`DaemonSetInfo`、`StatefulSetInfo`、`ReplicaSetInfo`
  - `JobInfo`、`CronJobInfo`
  - `NamespaceInfo`
- 新增 API 函数：
  - `listNamespaces(contextId)` - 获取所有命名空间
  - `listDeployments(contextId, namespace?)` - 支持 namespace 筛选
  - `listDaemonSets(contextId, namespace?)`
  - `listStatefulSets(contextId, namespace?)`
  - `listReplicaSets(contextId, namespace?)`
  - `listJobs(contextId, namespace?)`
  - `listCronJobs(contextId, namespace?)`

### 2. 更新 IPC 通信（index.ts + preload）
- 注册新的 IPC handlers
- 暴露新的 API 到渲染进程

### 3. 前端资源分类菜单栏
- 添加资源类型选择器（Tabs 或侧边菜单）
- 支持的资源类型：
  - 节点（Nodes）
  - 工作负载：Pod、Deployment、DaemonSet、StatefulSet、ReplicaSet
  - 任务：Job、CronJob

### 4. Namespace 筛选功能
- 顶部添加 Namespace 下拉选择器
- 支持"全部命名空间"选项
- 切换 Namespace 时自动刷新资源列表

### 5. 搜索和排序功能
- 搜索框：支持按名称模糊搜索
- 排序：点击表头进行升序/降序排序
- 搜索和排序在前端实现（不增加后端负担）

### 6. 样式优化
- 资源菜单栏样式
- 表格排序指示器
- 搜索框样式
